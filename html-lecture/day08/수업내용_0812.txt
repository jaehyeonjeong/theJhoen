레이아웃을 잡을 때 flex랑 grid를 많이 쓰는데
이 이둘의 차이점은 felx는 한줄일 때 사용
grid는 두 줄 이상인 경우 사용한다.


함수(function), 식(expression), 문(statement) 변수(variable)
함수에서는 함수명이 명확히 존재

  grid-template-columns: repeat(3, 1fr); // 1fr을 3번 반복
  grid-template-rows: repeat(3, minmax(100px, auto));
  // row의 높이를 적용할 떄 사용하는 minmax : 1번째 인자는 100px, max는 모름 자동
  // columns도 적용이 가능하나 보통은 잘 쓰지 않는다.

    // 휘하의 높이를 지정하는 방법이 두가지가 있음
  grid-template-rows: repeat(3, minmax(100px, auto));
  //grid-auto-rows: auto;

  z-index : 앞에서 겹쳐보이는 포지션
  grid garden 26번
  50px 0 0 0 1fr로도 가능(이미 세겨진 격자 한에서만 가능)
  fr(fraction) 또는 비율을 맞춰서 grid 비율을 조정할 수 있음
